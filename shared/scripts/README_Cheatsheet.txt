
# Zsh Alias Cheatsheet

## General Utilities
- `zs` → Edit `.zshrc` and reload configuration.
- `reload` → Reload Zsh without restarting.

## Neovim
- `vim` / `v` → Opens Neovim.

## Git Shortcuts
- `gs` → Short `git status`.
- `ga` → Add all changes.
- `gc "message"` → Commit with message.
- `gp` → Push changes.
- `gl` → Pretty git log.
- `gpl` → Pull latest changes.

## System Monitoring
- `htop` → Run htop with tree format.

## Fuzzy Finder (fzf)
- `fz` → Open fuzzy finder.
- `fzh` → Search command history.
- `fzg` → Search git log.

## Tmux
- `tns <name>` → Create a new session.
- `tls` → List sessions.
- `ta <name>` → Attach to a session.

## Terminal & File Management
- `term` → Open Alacritty terminal.
- `ls` → Enhanced `ls` using exa.
- `ll` → Detailed file listing.
- `lt` → Show directory tree.

## Grep & Search
- `grep` → Use `rg` instead of grep.

## Zip & Unzip
- `zipit <name>.zip <folder>` → Zip a folder.
- `unzipit <file>.zip` → Unzip file.

## Clipboard Management
- `copy` → Copy output to clipboard.
- `paste` → Paste from clipboard.

## Opening Files
- `open <file>` → Open file with default app.
